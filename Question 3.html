HTML (Hyper Text Markup Language)

HTML is a markup language. HTML stands for Hyper Text Markup Language. It helps in creating and structuring web pages. Without it the browsers wouldn't know what is the structure of your web page or even in what way to display the texts or images or any other thing in the web page.
HTML is an easy to use and learn language. It also allows the user to embed multimedia content like images and videos in the web page. In general HTML is used with seamlessly integration with CSS and JS for making webpages and mobile applications.
The file extension for an HTML file is ".html"
